# DermEval

DermEval is the reference-free, case-level evaluator paired with DermBench. Given a dermatology image and a candidate diagnostic narrative, it produces a structured critique plus six 0-5 ratings:

1. Accuracy
2. Safety
3. Medical Groundedness
4. Clinical Coverage
5. Reasoning Coherence
6. Description Precision

## Included Code

The release keeps DermEval as a compact subpackage under `dermbench.dermeval`:

- `constants.py`: canonical dimension names, aliases, and compact rubrics.
- `prompts.py`: prompt construction and canonical evaluation text helpers.
- `score_parser.py`: deterministic local parsing of six dimension scores.
- `dataio.py`: JSON/JSONL loading, record validation, and score normalization.
- `convert_to_llava_sft.py`: conversion from raw DermEval records to LLaVA-style SFT conversations.
- `llava_infer.py`: optional local inference entrypoint for compatible DermEval/LLaVA checkpoints.

Model weights, training outputs, private labels, and image files are not bundled in this repository.

## Record Schema

DermEval records can be stored as JSONL or as a JSON list. Image paths should be relative to the local image root used by the runner.

```json
{
  "id": "case_001",
  "image": "relative/path/to/image.jpg",
  "diagnostic_text": "Candidate diagnostic narrative generated by a model.",
  "evaluation_text": "Accuracy: 4.0 - ...\nSafety: 5.0 - ...",
  "scores": {
    "accuracy": 4.0,
    "safety": 5.0,
    "medical_groundedness": 4.0,
    "clinical_coverage": 3.5,
    "reasoning_coherence": 4.0,
    "description_precision": 4.0
  }
}
```

`evaluation_text` is used as the supervised target when present. If it is absent, the converter builds a canonical target from `scores` and optional per-dimension rationales.

## Basic Utilities

Build LLaVA-style supervised data:

```bash
python -m dermbench.dermeval.convert_to_llava_sft \
  --input data/dermeval_records.jsonl \
  --output outputs/dermeval_sft.json
```

Parse scores from a generated critique:

```bash
python -m dermbench.dermeval.score_parser path/to/dermeval_output.txt
```

Run local single-case inference when a compatible checkpoint and LLaVA-style runtime are available:

```bash
pip install -e ".[dermeval]"
python -m dermbench.dermeval.llava_infer \
  --model-path path/to/dermeval_or_llava_checkpoint \
  --image-file path/to/image.jpg \
  --diagnostic-text-file path/to/candidate_narrative.txt \
  --conv-mode llava_v1 \
  --temperature 0
```

The inference command returns JSON with the critique text and parsed six-dimensional scores.
